* Changelog: Cursor

All notable changes to *Cursor* are documented in this file.

** [2.0.0] 2026

*** Added

- ~CHANGELOG.org~, which contains notable changes to *Cursor*.

- Directory ~unicode~, which contains lookup tables for grapheme cluster segmentation.

*** Changed

- Scheme strings and characters are now base types of PEG expressions. They no
  longer need to be wrapped in ~text~ or ~char~ constructors. ~text~ and ~char~
  are no longer part of the public API.

- ~(capture px)~ and ~(capture function px)~ operate over ~(substring string start end)~
  instead of ~(list char ...)~.

- ~(transform function px)~: Restricted ~function~ scope to captures and transformations
  within the pattern ~px~. Captures and transformations are passed directly as arguments,
  where before all arguments were passed through a shared accumulator with an optional stack
  discipline.
